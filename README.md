# WhisperFlow 🎙️

Персональный голосовой ввод для macOS.  
Нажми **Fn + Пробел** → говори → текст вставляется туда, где стоит курсор.

Работает на **Groq API** (Whisper + Llama) — быстро и бесплатно.

---

## Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/timaskar/wispr-flow.git
cd whisperflow
```

### 2. Создать виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Получить API-ключ Groq (бесплатно)

1. Зайдите на [console.groq.com](https://console.groq.com)
2. Зарегистрируйтесь (бесплатно)
3. Перейдите в **API Keys** → **Create API Key**
4. Скопируйте ключ (начинается с `gsk_...`)

### 4. Запустить

```bash
python whisperflow.py
```

При первом запуске приложение попросит вставить API-ключ в диалоговое окно.  
Ключ сохранится локально в `~/.config/whisperflow/config.json`.

---

## Использование

| Действие | Клавиша |
|---|---|
| Начать / остановить запись | **Fn + Пробел** |
| Отменить запись | **Esc** |

Текст автоматически вставляется в то место, где стоит курсор — работает в любом приложении.

---

## Индикатор

Внизу экрана по центру — плавающий индикатор:

- **Серая точка** — готов к записи
- **Тёмная полоса с волной** — идёт запись
- **Пульсация** — распознавание
- **Зелёный ✓** — текст вставлен
- **Красная корзина** — запись отменена

---

## Настройки

Файл конфигурации: `~/.config/whisperflow/config.json`

```json
{
  "groq_api_key": "gsk_...",
  "language": "ru",
  "post_processing": true,
  "proxy_sub_url": ""
}
```

| Параметр | Описание |
|---|---|
| `groq_api_key` | Ваш ключ Groq API |
| `language` | Язык распознавания: `"ru"`, `"en"`, `"auto"` |
| `post_processing` | AI-очистка текста: убирает слова-паразиты, расставляет пунктуацию |
| `proxy_sub_url` | Ссылка подписки на прокси (для РФ, где Groq заблокирован) |

## Разрешения macOS

При первом запуске macOS попросит два разрешения:

1. **Микрофон** — появится автоматически, нажмите «Разрешить»
2. **Accessibility** — нужно включить вручную:
   - **System Settings → Privacy & Security → Accessibility**
   - Найдите Python (или Terminal) и включите тумблер
   - Перезапустите WhisperFlow

> Без Accessibility не будет работать горячая клавиша и вставка текста.

## Прокси (для России)

Groq API заблокирован в РФ. При первом запуске приложение попросит вставить ссылку подписки на прокси (Shadowrocket / V2Ray / sing-box).

Поддерживаемые протоколы: Shadowsocks, Trojan, VLESS, VMess.

Для работы прокси нужен установленный [sing-box](https://github.com/SagerNet/sing-box):

```bash
brew install sing-box
```
